# Applied research: agent-classes

Tracks the [agent-classes](https://github.com/usebessemer/agent-classes) library: the vertical agent-employee classes, typed by the [agent-role topology](../../theory/agent-role-topology.md) and scaffolded by icm-kit.

## Current

The first class is the **Bookkeeper** (Executor × Standing, vertical-agnostic), at charter v0.1. The open research question is the leverage thesis itself: is a class genuinely *reusable* (one charter many businesses instantiate) or bespoke with extra steps? The method is **instance-first**: build the first instance concretely, then extract what is reusable (the class) from what is bespoke (the instance). The seam only validates at instance #2.

Cycle folders are added as the cadence formalizes; see [../README.md](../README.md) for the conventions.
