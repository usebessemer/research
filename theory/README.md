## How to add a piece

1. Branch `feature/theory-<slug>` off main.
2. Draft `theory/<slug>.md` (or `theory/<slug>/main.md` if supporting files are required).
3. Iterate on the branch.
4. When ready, open a pull request against main, secure review, and merge to publish.
5. Add a row to the index above.
