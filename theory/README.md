# Theory

Free-standing investigations into programming patterns, emerging technology, methodologies, and adjacent ideas. Each piece is contained in its own file, or in its own folder if it requires supporting material. Drafts are developed on feature branches; merging to main constitutes publication.

## How to add a piece

1. Branch `feature/theory-<slug>` off main.
2. Draft `theory/<slug>.md` (or `theory/<slug>/main.md` if supporting files are required).
3. Iterate on the branch.
4. When ready, open a pull request against main, secure review, and merge to publish.
5. Add a row to the index above.
